# StreamVLN-Comp

This bundle was organized from `videollm-comp-on-streamvln-detailed-report.md`.

Directory layout:

- `VideoLLM-Comp-Training-Free on StreamVLN`: migrated VideoLLM-Comp compression methods and closed-loop evaluation summaries.
- `VideoLLM-Comp-Training-Aware on StreamVLN`: reserved for heavier VideoLLM-Comp variants that are not part of the current migrated upload.
- `Others`: the internal `f2_warmup_eval_target_hw7` line, including related pruning modules, training patch files, AR-only evaluation summaries, and a runnable AR-only eval script.

Only the main closed-loop evaluation materials are included. Auxiliary diagnostic tables, plots, and scripts were intentionally left out for now.
